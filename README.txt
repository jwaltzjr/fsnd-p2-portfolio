Udacity's FSND Portfolio
(Project 2)


Description
-----------

This grunt packgage creates a portfolio website of projects completed by Jeff Waltz for Udacity's Full Stack Web Development Nanodegree.


Files
------------------

Gruntconfig.js - A javascript file that uses node.js to verify, combine, and minfy html, css, and js documents, as well as minify images and covert scss files to a single css file for production.

Gruntconfig.yml - A config file that hosts the directories of the source files for grunt to process, as well as the distribution directories for the production website.

packgage.json - A json object containing a description of this grunt packgage as well as all of the development dependencies.

/src - Source files, including all images, all html/scss/js scripts, and all third-party frameworks and fonts.

/dist - Production directory, which contains the final rendered site.
